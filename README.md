Project Overview :

The Netflix Performance Analysis project aims to analyze Netflix data to understand content distribution trends performance patterns and platform growth insights. The objective is to transform raw data into meaningful information that can support content strategy and business decisions.

This project follows an end to end data analysis workflow using SQL and Excel.

Tools and Technologies Used :

SQL for querying ,analysis and aggregations Microsoft Excel for data cleaning pivot tables charts and dashboards GitHub for documentation and version control


The dataset is used for learning and analytical purposes only.

Key Analysis Performed :

Data cleaning and formatting using Excel SQL based analysis of content distribution Movie and TV show performance comparison Genre wise and country wise analysis Year wise content trend analysis

KPIs and Insights :

Total Movies and TV Shows Top Genres by Content Count Country wise content distribution Year wise content growth trend Content type contribution to platform

These insights help understand how Netflix content is distributed and how the platform has evolved over time.

Excel Dashboard Overview :

Pivot tables for content and genre analysis Charts for trend and distribution visualization Country wise performance comparison Simple and business friendly dashboard

Business Recommendations :

Focus on high demand genres Increase content investment in growing regions Balance movie and series production Use trend analysis for future content planning
